# Privacy Policy

Note Database is a local-first Obsidian plugin.

- The plugin does not send notes, frontmatter, formulas, database configuration, or usage data to any external server.
- The plugin reads Markdown files, `.base` files, and Obsidian metadata inside the current vault only to render database views and apply user-requested edits.
- The plugin writes only to the local vault through Obsidian APIs, including frontmatter updates, generated database files, and plugin settings.
- No telemetry, analytics, remote logging, or network requests are used by the plugin.
